Alt_R키를 단축키로 사용하는 프로그램(Chrome)등에서는 동작이 안됨 Ctrl_R은 Hanja 키로 매핑 시키고 싶이 않음. Use Xmodmap Hangul key xmodmap -e 'remove mod1 = Alt_R' # Alt_R의 기본 키 매핑 제거 xmodmap -e 'keycode 108 = Hangul' # Alt_R을 Hangul 키로 매핑 Hanja Key xmodmap -e 'remove control = Control_R' xmodmap -e 'keycode 105 = Hangul_Hanja' Save permanently xmodmap -pke > ~/.
